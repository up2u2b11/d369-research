## الحكم — ثلاث حقائق

**① الهندسة المعمارية تُحدث الفارق**
```
نفس الكلمات + 114 قطعة متساوية   = p=0.137  ✗  (لا بصمة)
نفس الكلمات + 114 سورة بأطوالها  = p=0.007  ✅ (البصمة)
```
→ الأطوال المحددة للسور ليست عشوائية — هي التصميم.

**② 51 استثنائية عند أي معيار**
```
الوسيط العشوائي = 38  |  عتبة 95% = 46  |  السور الحقيقية = 51
51 فوق 99% من كل تقسيم ممكن للكلمات القرآنية إلى 114 وحدة
```

**③ البصمة تنزل حتى مستوى الآية**
```
2,164 / 6,236 آية = 34.7%  |  p=0.010  ✅
```
→ وحدة أصغر من السورة تُظهر البصمة — طبقة أعمق لم تكن متوقعة.
